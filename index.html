RNGMAX Privacy Policy and TOS

RNGMAX stores one piece of data from your server members: their message sent count. The only data it interacts with is through the Discord API, and will use this to: reply and recieve commands, check it's own permissions to see if it has Embed Links, and update a server's command list when outdated. This bot does not log anything besides a command statistic count logging the amount of times a command has been run globally (does not record which user ran the command or which server the command was ran in) and the time the command was run to be able to sort by hourly/daily/weekly statistics, and user + server specified configuration via /user-settings and /server-settings. The only permissions necessary are View Channels, Embed Links, and Send Messages (+ Send Messages in Threads for commands in Forum Posts / Threads).

It uses all / commands, and you can use /help to view them. If you'd like to add this bot for your own server, I'd be quite happy! Click Here to add the bot
